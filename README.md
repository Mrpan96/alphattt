Alpha Tic Tac Toe Ultimate
=====


About
-----

Alpha Tic Tac Toe Ultimate(下称 AlphaTTT)项目是一个开源的棋类在线对战平台项目。支持人工智能程序（下称AI）和人类同时接入并且对弈，能够提供排名和观战功能。

本项目包括一个使用[Yaws](http://yaws.hyber.org)框架实现的Web服务器，以及一个使用[Erlang](http://www.erlang.org/downloads)语言实现的对战服务框架，因而对战平台主要使用了Erlang语言实现。

Install
-----

本项目依赖[Yaws](http://yaws.hyber.org)和[Erlang](http://www.erlang.org/downloads)。
请根据您的操作系统，下载Yaws1.97以后版本，Erlang的Otp16R以后版本。

请先clone本项目代码到本地。然后,

'''
cd src
erl -make
'''
